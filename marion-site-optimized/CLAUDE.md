# Projet Marion Balsaux — Site web
## Stack
HTML/CSS/JS vanilla · Déployé sur Netlify · Repo GitHub : marion-site

## Palette de couleurs
- Beige : #F5F0E8
- Dark : #1C1A17
- Gold : #B5986A
- Gold light : #D4B98A

## Typographies
- Cormorant Garamond (serif) → titres
- DM Sans (sans-serif) → corps de texte

## Structure des fichiers
- index.html → page d'accueil
- projets.html → portfolio
- services.html → offres
- approche.html → méthode
- contact.html → formulaire
- styles.css → tous les styles
- main.js → interactions

## Règles importantes
- Ne jamais changer le design global sans demander
- Corriger uniquement ce qui est demandé
- Toujours tester en local (Live Server) avant de proposer
- Respecter l'alternance des cartes services (gauche/droite)
- Le logo est dans images/site/logo.png
